# Отчёт по лабораторной работе №3

Выполнил: Малинин Ярослав Юрьевич, группа М8О-102БВ-25

## Цель работы
Реализовать вычисление арифметических выражений с использованием дерева выражений. Удалить из дерева выражений слагаемые равные нулю.

## Структура проекта
- `src/` - содержит исходный код программы
- `src/stack.c/h` - модуль стека операторов
- `src/tree.c/h` - модуль дерева выражений
- `src/tree_stack.c/h` - модуль стека узлов дерева
- `src/tokenization.c/h` - модуль токенизации выражений
- `src/validation.c/h` - модуль валидации выражений
- `src/calculator.c/h` - модуль вычисления дерева выражений
- `tests/` - содержит тесты для модулей

## Содержание работы
- `main.c` - основной модуль программы, содержит функции для считывания выражения, токенизации, перевода в постфиксную запись, построения дерева выражений, рефакторинга дерева и вычисления результата
- `stack.c/h` - реализация стека операторов для преобразования в постфиксную запись
- `validation.c/h` - валидация входного выражения (проверка скобок, допустимых символов, повторяющихся операторов)
- `tokenization.c/h` - токенизация выражения и преобразование в постфиксную запись
- `tree.c/h` - построение дерева выражений из постфиксной записи, вывод и рефакторинг дерева
- `tree_stack.c/h` - реализация стека узлов дерева для построения выражения
- `calculator.c/h` - вычисление значения дерева выражений

## Команды для запуска

```bash
make run # сборка программы
./run # запуск программы
make all_tests # сборка тестов
./all_tests # запуск тестов
make clean # удаление объектных файлов и исполняемых файлов
```
