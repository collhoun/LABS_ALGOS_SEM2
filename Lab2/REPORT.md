# Отчёт по лабораторной работе №2

Выполнил: Малинин Ярослав Юрьевич, группа М8О-102БВ-25

## Цель работы
Реализовать метод сортировки двоичной вставкой на таблице на строковом типе ключа, данные хранятся вместе

## Структура проекта
- `data_samples/` - содержит данные для работы программы
- `src/` - содержит исходный код программы
- `src/data_creators/` - содержит скрипты для создания данных
- `tests/` - содержит тесты и скрипт для создания тестиружных данных
- `tests_data/` - содержит данные для тестов

## Содерание работы
- `main.c` - основной модуль программы, содержить функции для считывания данных, сортировку двочиными вставками,бин поиск, вывод результатов сортировки и демонстрации работы программы
- `main.h` - заголовочный файл для импорта и тестирований

## Команды для запуска

```bash
make data_samples # создание исходных данных
make data_tests # создание тестовых данных
make test # запуск тестов
make run # сборка программы
./main # запуск программы
make clean_tests # удаление тестовых данных
make clean # удаление  main.exe
```
