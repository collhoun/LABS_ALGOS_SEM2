## Цель работы
Реализовать красно-черное дерево с поддержкой следующих операций:
1. Добавление узла к дереву по ключу (с балансировкой при необходимости).
2. Удаление узла из дерева по ключу (с балансировкой при необходимости).
3. Печать дерева.
4. Поиск значения элемента по заданному ключу.
Для красно-черного дерева предусмотреть при выводе на печать маркировку цвета узла (с помощью разных видов скобок).
Предусмотреть возможность ввода команд из текстового файла в формате <номер операции> [<ключ>] [<значение>].
Предусмотреть также вывод в текстовый файл в формате:
<команда из входного файла>
<её результат>
…
<команда из входного файла>
<её результат>

## Структура проекта
- `src/` - содержит исходный код программы
- `src/tree.c/h` - модуль красно-черного дерева
- `src/main.c` - основной модуль программы
- `src/data_loader.py` - скрипт для генерации тестовых данных
- `tests/` - содержит тесты для модулей
- `data/` - содержит входные и выходные файлы
- `случаи/` - содержит диаграммы случаев вставки и удаления

## Содержание работы
- `main.c` - основной модуль программы, содержит функции для считывания команд из файла, выполнения операций с красно-черным деревом и вывода результатов
- `tree.c/h` - реализация красно-черного дерева: вставка узла с балансировкой, удаление узла с балансировкой, поиск значения по ключу, печать дерева с маркировкой цвета узлов
- `data_loader.py` - скрипт для генерации тестовых данных
- `tests/test.c` - модульные тесты
- `tests/test_tree_many.c` - тесты для большого количества операций

## Команды для запуска

```bash
make all # сборка программы
./main # запуск программы
make tests # сборка тестов
./test # запуск тестов
make clean # удаление исполняемых файлов
```

## Замечание
- поддерживаемые операции: вставка (1), удаление (2), печать дерева (3), поиск (4)
- ключи - строки длиной до 6 символов, значения - целые числа
- цвет узлов при печати маркируется с помощью разных скобок: красные - [], черные - ()
- входные команды читаются из файла `data/samples.txt`, результаты выводятся в `data/output.txt`
