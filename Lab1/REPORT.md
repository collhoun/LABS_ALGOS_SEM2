## Цель работы
Реализовать дерево с базовыми операциями, дополнительно реализовать метод нахождения ширины __произвольного__ дерева

## Содержание
- `trees.c` — реализация структуры дерева, списка потомков и операций
- `trees.h` — заголовочный файл для импорта и тестирований
- `test.c` — тесты
- `Makefile` — compose на минималках (сборочный файл)

## Как собрать и запустить
В каталоге `Lab1` выполнить:

```bash
make all
.\main
```


Запуск `main` выполняет несколько тестов из `test.c`. Пятый тест намеренно завершается ошибкой.

## Реализованные структуры и операции
- Каждый узел дерева (`TreeNode`) содержит поле `list_of_children`, реализованное как односвязный список `List` элементов типа `Node`.
- Операции:
  - `add_in_tree()` — добавление узла в дерево (рекурсивно ищет место по `max_children`).
  - `delete_node()` — удаление узла по ключу; удаляет поддерево и освобождает память.
  - `print_tree()` — печать дерева в консоль с отступами по глубине.
  - `count_width_tree()` — вычисление ширины дерева (максимального числа узлов на одном уровне).

## Тесты (в `test.c`)
- TEST_1: проверка добавления узлов
- TEST_2: проверка удаления одного узла
- TEST_3: проверка подсчёта ширины дерева
- TEST_4: построение линейного списка (каждый узел имеет максимум 1 потомка) и проверка ширины
- TEST_5: попытка удаления корня (ожидаемое завершение программы с ошибкой)

## Замечания
- Тест `TEST_5` намеренно завершает программу.
